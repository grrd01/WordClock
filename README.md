# grrd's WordClock

This repository contains the code for the LOLIN (WEMOS) D1 mini Lite (ESP8266) Wordclock offered by FabLab Bern:
[WordClock-Worshop @ FabLab Bern](https://fablab-bern.ch/wordclock-workshop/)
